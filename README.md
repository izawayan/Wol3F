# yfinance-br-quant 📊🇧🇷

Um wrapper robusto e otimizado do `yfinance`, focado no tratamento de dados para o mercado financeiro brasileiro (B3). 

Ao lidar com dados quantitativos de ações brasileiras, é comum encontrar problemas com ativos deslistados, suspensos ou com falhas de histórico. Esta biblioteca resolve esses gargalos, garantindo pipelines de dados mais seguros para análises quantitativas e backtesting.

## 🚀 Instalação

Você pode instalar a biblioteca diretamente do PyPI usando o pip:

```bash
pip install yfinance-br-quant
